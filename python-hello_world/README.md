# 0x00. Python - Hello, World

## Description

This project is an introduction to Python programming, covering basic
Shell scripting integration with Python, variable usage, string
manipulation, and simple I/O with the `print` function.

It is part of the ALU Higher Level Programming curriculum.

## Repository

- **GitHub repository:** `alu-higher_level_programming`
- **Directory:** `python-hello_world`

## Requirements

- All scripts interpreted/compiled on Ubuntu 20.04 LTS
- Python scripts: first line must be `#!/usr/bin/python3`
- Shell scripts: first line must be `#!/bin/bash`
- All files must end with a new line
- All Python files must be executable (`chmod +x filename`)
- Code should follow the pycodestyle style guide (version 2.8.*)
- Length of files will be tested using `wc`

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-run` | Shell script that runs a Python file given in `$PYFILE` |
| 1 | `1-run_inline` | Shell script that runs Python code given in `$PYCODE` |
| 2 | `2-print.py` | Prints `"Programming is like building a multilingual puzzle` |
| 3 | `3-print_number.py` | Prints an integer followed by "Battery street" using f-strings |
| 4 | `4-print_float.py` | Prints a float with 2 digits of precision using f-strings |
| 5 | `5-print_string.py` | Prints a string 3 times, then its first 9 charac:;ters |
